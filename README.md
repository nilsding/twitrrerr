# twitrrerr

A simple Twitter client.

## Requirements

* Ruby 2.0+
* Bundler

## Usage

First of all, install all dependencies and convert the UI files.

    # This may take forever depending on your computer and your installed gems, so be patient!
    $ bundle
    # Convert UI files
    $ rake build_ui

Finally, run Twitrrerr:

    $ bin/twitrrerr
